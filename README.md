# LLM Chat Web版 (v0.1)

把 [LLM Chat](https://github.com/guobinqiu/llm-chat) (命令行版) 改成 Web 版

## 运行

1. 运行后端服务

```
cd backend && go run main.go
```

2. 运行前端服务

```
cd frontend && npm run serve
```
